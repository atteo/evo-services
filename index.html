Moonshine

Guice-based Java application framework

  • Twitter
  • Github
Loose coupling

Individual parts of the application are clearly separated and know very little about each other

Bundled services

Many common Java technologies like Hibernate, Jetty and others are already available as Moonshine services

Configuration

XML configuration files allow to quickly select desired functionalities

Flexibility

Quickly select desired functionalities whether you need single REST service or a full-fledged multitier environment

Testability

Easily start the Moonshine container as a part of your test

Apache license

Free to use for everyone, for both commercial and non-commercial uses

Bootstrap your app in seconds

Execute

mvn archetype:generate \
	-DarchetypeGroupId=org.atteo.moonshine \
	-DarchetypeArtifactId=service-archetype

Select the services

Choose what functionalities you require

<dependency>
	<groupId>org.atteo.moonshine</groupId>
	<artifactId>jetty</artifactId>
</dependency>
<dependency>
	<groupId>org.atteo.moonshine</groupId>
	<artifactId>hibernate</artifactId>
</dependency>
